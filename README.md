# File to C array converter
Coverts any file to a C style array.
Useful if you want to embed a file (binary, text, image, whatever) into your code!
Use it for your Arduino or other embedded projects.

### Forked from this [original repo](https://github.com/notisrac/FileToCArray) with some small tweaks:
- Restyled and reorgnised the website for a dark-mode theme and clean design
- Added ability to rename the C-style array and the `#define` statements
- Added a preset option for my usecase (using a ST7735 display driver)
  - presets are editable in `line 343 of main.js` and `line 84 of index.html`

All credit goes to **notisrac** and his fantastic project
